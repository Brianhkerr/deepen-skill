# Examples

A **real `/deepen` run**, committed verbatim as the hero example — so you can see actual output (a source
corpus, falsifiable stances, track-record-weighted experts, a ranked playbook) rather than a mockup.

> Per the skill's own honesty rule, this is a genuine run with real, cited sources — never a fabricated
> sample. The load-bearing figures were re-verified against primary records (see each `sources/` digest's
> `verified:` line).

## [`deliberate-practice/`](./deliberate-practice/)

`/deepen deliberate practice` — chosen because it has a live, *checkable* expert disagreement (Ericsson's
deliberate-practice theory vs. the Macnamara/Hambrick meta-analyses) that showcases the skill's signature
moves: weighting sources by track record, taking falsifiable stances, and red-teaming its own conclusions.

Start with [`_index.md`](./deliberate-practice/_index.md) — the one-context map with the depth dashboard
and a "to do X, read these files" router — then read [`disagreements.md`](./deliberate-practice/disagreements.md),
the highest-value file, where the skill takes positions on the open controversies and states exactly what
evidence would flip each one.

| File / folder | What it shows |
|---|---|
| [`_index.md`](./deliberate-practice/_index.md) | The loadable map: scope, depth dashboard, read-first router, aliases |
| [`sources/`](./deliberate-practice/sources/) | The corpus — 8 high-fidelity source digests, each with a verified-against-primary line |
| [`principles.md`](./deliberate-practice/principles.md) | Durable findings, each with a "common misconceptions" line + cite-or-flag tags |
| [`experts.md`](./deliberate-practice/experts.md) | Sources ranked by track-record proxy ladder, not fame |
| [`disagreements.md`](./deliberate-practice/disagreements.md) | Crux · stance · confidence · falsifier · survived-refutation (with claim-IDs) |
| [`playbook.md`](./deliberate-practice/playbook.md) | ICE-ranked learner tactics tied back to the principles |
| [`failure-modes.md`](./deliberate-practice/failure-modes.md) | The expensive anti-patterns — the errors experts are defined by NOT making |
| [`benchmarks.md`](./deliberate-practice/benchmarks.md) | The reference numbers, each linked to its source digest |
| [`frontier.md`](./deliberate-practice/frontier.md) | Open unknowns + where to dig next |
| [`glossary.md`](./deliberate-practice/glossary.md) | The construct vocabulary, doubling as a retrieval bridge |
| [`changelog.md`](./deliberate-practice/changelog.md) | Accretion log — how each run deepens it |

**Note the honesty seams on purpose:** the run holds one debate (does practice *quality* dominate
quantity?) at explicitly LOW confidence and flags it as the thinnest area for a future pass — a scoped,
labeled non-answer rather than a confident guess. Several `sources/` digests also state plainly what they
*can't* support (e.g. paywalled full texts not fetched).
