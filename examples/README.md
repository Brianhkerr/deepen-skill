## [`deliberate-practice/`](./deliberate-practice/)

`/deepen deliberate practice` — chosen because it has a live, *checkable* expert disagreement
(Ericsson's deliberate-practice theory vs. the Macnamara/Hambrick meta-analyses) that showcases the
skill's signature moves: weighting sources by track record, taking falsifiable stances, and red-teaming
its own conclusions.

Start with [`_map.md`](./deliberate-practice/_map.md) — the index + depth dashboard — then read
[`disagreements.md`](./deliberate-practice/disagreements.md), the highest-value file, where the skill
takes positions on the open controversies and states exactly what evidence would flip each one.

| File | What it shows |
|---|---|
| [`_map.md`](./deliberate-practice/_map.md) | Scope, depth-calibration dashboard (solid vs. thin), aliases |
| [`principles.md`](./deliberate-practice/principles.md) | Durable findings, each with a "common misconceptions" line + cite-or-flag tags |
| [`experts.md`](./deliberate-practice/experts.md) | Sources ranked by track-record proxy ladder, not fame |
| [`disagreements.md`](./deliberate-practice/disagreements.md) | Crux · stance · confidence · falsifier · survived-refutation |
| [`tactics.md`](./deliberate-practice/tactics.md) | ICE-ranked learner playbook tied back to the principles |
| [`changelog.md`](./deliberate-practice/changelog.md) | Accretion log — how a re-run would deepen it |

**Note the honesty seams on purpose:** the run holds one debate (does practice *quality* dominate
quantity?) at explicitly LOW confidence and flags it as the thinnest area for a future pass — a scoped,
labeled non-answer rather than a confident guess.
